# RobotArm

# Additional details used in these parts:

- 3mm bolts and nuts

- 693ZZ bearings (3x8x4mm) and 623ZZ (3x10x4mm)

- LX16A smart servos

- 2mm pitch timing belt (GT2)

- 6mm airsoft BB (plastic) balls for base bearings
